You're in the Info.plist file in Xcode, which is where you need to add permissions for speech recognition and microphone access.

Steps to Add Permissions in Info.plist
Click the + button next to the last key in the list.
In the Key field, type:
NSSpeechRecognitionUsageDescription
In the Type column, select String.
In the Value column, enter:
"We need speech recognition to process voice requests."
Repeat the steps to add another key:
NSMicrophoneUsageDescription
Set Type: String
Set Value: "We need microphone access to record your speech."
Final Info.plist Should Contain:
xml
Copy
Edit
<key>NSSpeechRecognitionUsageDescription</key>
<string>We need speech recognition to process voice requests.</string>

<key>NSMicrophoneUsageDescription</key>
<string>We need microphone access to record your speech.</string>
6. Save & Close
